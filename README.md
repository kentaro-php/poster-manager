# ポスター管理ツール

街角に貼っている政党告知ポスターの管理ツール。スタッフ・支援者と共有して使用。

## 機能

- 📊 **ダッシュボード**: 進捗・ステータス分布・担当者ランキング・更新ログ
- 📝 **一覧 + フィルタ**: 検索・ステータスで絞り込み
- 🗺️ **地図表示**: ステータス別色分け、現在地表示、ナビ起動
- ✏️ **CRUD**: スマホから現地で追加・編集・削除
- 📷 **写真添付**: 各ポスターに証拠写真を保存
- 📥 **CSVインポート**: Excelで作ったリストを一括取り込み
- 🌓 **ダーク/ライト自動切替**

## 構成

```
[GitHub Pages: HTML/CSS/JS]
         ↓ ↑
   ┌─────────────┐
   ↓             ↓
[Apps Script]  [Cloudflare Worker]
   ↓             ↓
[スプレッドシート]  [R2: 写真ストレージ]
```

- **フロントエンド**: GitHub Pages（無料）
- **データ保存**: Google スプレッドシート（無料）
- **API**: Google Apps Script Web App（無料）
- **写真**: Cloudflare R2 + Worker（無料枠）

## セットアップ手順

詳細は [SETUP.md](SETUP.md) を参照。

1. Google スプレッドシートを作成
2. Apps Script を貼り付けてWebアプリとしてデプロイ
3. Cloudflare R2 バケットを作成
4. Cloudflare Worker（写真アップロード）をデプロイ
5. このプロジェクトを GitHub Pages に公開
6. 公開ページの初回アクセス時に各URLを設定

## アクセス制御

- 公開URLを知っている人だけアクセス可能（簡易方式）
- 書き込み時に必ず **スタッフ名** が記録される
- スプレッドシート編集履歴は Google が自動記録
